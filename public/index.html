Instagram sessionid 쿠키 찾는 방법

  1. 브라우저에서 Instagram에 로그인
  2. F12 눌러서 개발자 도구 열기
  3. Application 탭 클릭 (Chrome) 또는 Storage 탭 (Firefox)
  4. 왼쪽에서 Cookies → https://www.instagram.com 선택
  5. sessionid 찾아서 Value 복사
